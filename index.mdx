## Getting Started

This CMS is designed for personal use with simplicity in mind. All content is stored as markdown files that you can edit directly.

<Steps>
  <Step>
    **Create Content**: Use the "Create New Content" card above or add `.mdx` files directly to the content folders
  </Step>
  
  <Step>
    **Organize**: Content is automatically organized by folder structure
  </Step>
  
  <Step>
    **Edit**: Click the edit button on any page to modify content directly
  </Step>
</Steps>

<Note>
This is a private CMS - content is only accessible to you and is not published publicly unless you choose to share it.
</Note>
